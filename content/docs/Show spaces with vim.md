# Show spaces with vim
```
:set listchars=eol:$,tab:>-,trail:~,space:␣
:set list
```

The `space:␣` is gold. Thank you [loxaxs](https://askubuntu.com/questions/74485/how-to-display-hidden-characters-in-vim#comment1621885_74503)!
